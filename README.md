# 📚 C++ 알고리즘 문제 풀이 환경

이 프로젝트는 여러 개의 C++ 소스 파일(`exam_code` 폴더)을 하나의 공통 환경(`main.cpp`)에서 효율적으로 빌드하고 테스트하기 위한 환경입니다.

---

## 📂 폴더 구조

.
├── main.cpp            # 공통 실행 로직 및 헤더 포함 (고정)
├── Makefile            # 빌드 자동화 스크립트
├── app.exe             # 생성된 실행 파일 (빌드 후 생성)
└── exam_code/          # 각 문제의 소스 코드가 위치하는 폴더
    ├── 2557.cpp
    ├── 1000.cpp
    └── ...

---

## 🛠️ 개발 환경 및 설정

* **Compiler**: `g++` (GCC)
* **Standard**: C++17
* **Encoding**: 
  * 소스 코드: `UTF-8`
  * 실행 환경: Windows CMD (`CP949`) 대응 설정 완료
* **Build Tool**: `make` (Windows용)

---

## 🚀 사용 방법

터미널(CMD)에서 프로젝트 루트 폴더로 이동한 뒤, 문제 번호를 입력하여 빌드합니다.

### 1. 문제 빌드 (Build)
`make` 뒤에 실행하고자 하는 **문제 번호**를 입력하세요. (확장자 `.cpp`는 생략합니다.)

```bash
make 2557
